Wrap Jekyll Theme

Documentation by Andy Lu

Thanks for buying "Wrap", a creative responsive Jekyll theme for many purposes, it could be used as company, design studio, business or portfolio websites.

Installation

          
  ~ $ gem install jekyll bundler
  ~ $ cd wrap
  ~/wrap $ bundle install
  ~/wrap $ bundle exec jekyll serve
  # => Now browse to http://localhost:4000

HTML Structure

This template is based on foundation framework, to understand what the "row" and "column" classes mean, please refer to the foundation documentation. 

          <!--

  HTML files:

  _layouts:
    1. blog.html -> the blog layout file
    2. default.html -> the default onepage layout file
    3. post.html -> blog post layout file
    4. work.html -> portfolio item single page layout

  blog:
    1. index.html -> the blog index page

-->

Contact form

The contact form uses formspree.io service to actually send the email, you can configure the recipient email address by setting the "email" variables in _config.yml, the contact-form code is in file "_includes/contact-form.html".

Commenting system

The commenting system uses Disqus service to record the comments, you can set it up by setting the "disqus.shortname" variable in _config.yml, the related code is "_includes/disqus_comments.yml".
